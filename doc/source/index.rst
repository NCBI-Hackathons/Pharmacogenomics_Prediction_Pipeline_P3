Pharmacogenomics Prediction Pipeline (P3)
=========================================
Overview
--------

The P3 pipeline aims to predict drug sensitivity utilizing second-generation
sequencing data and biological annotation (gene ontology, pathways, etc.), and
*in vitro* high-throughput drug screening data.
